# My Portfolio (use Numble)

## Spec
### 1. Framework
- Next.js 13
### 2. Language
- Typescript 5.1.3
### 3. Package manager
- yarn berry
### 4. CSS Library & Framework
- Framer motion
- Tailwindcss
### 5. CI
- husky
- Vercel
### 6. Deploy
- Vercel

## Installation
```bash
$ yarn
```

## Lint
```bash
# Check code quality
$ yarn lint
# Fix code quality
$ yarn lint:fix
```

## Usage
### Dev
```bash
$ yarn dev
```
### Build
```bash
$ yarn build
```
### Start Prod
```bash
$ yarn start
```
